## Smart Resume Analyzer

The **Smart Resume Analyzer** is an AI-powered web application that helps job seekers evaluate and improve their resumes by comparing them with job descriptions. The system analyzes resume content using Natural Language Processing (NLP) and Machine Learning techniques to provide insights such as skill matching, missing keywords, and resume scoring.

This project aims to simplify the recruitment process by enabling candidates to optimize their resumes according to industry requirements and specific job roles.

### Key Features

* Resume upload (PDF/DOCX)
* Automatic text extraction from resumes
* Skill and keyword extraction using NLP
* Job description matching
* Resume score calculation
* Suggestions for improvement
* User-friendly interface built with **Streamlit**

### Tech Stack

* **Python**
* **Streamlit** – Frontend UI
* **Natural Language Processing (NLP)**
* **Scikit-learn**
* **NLTK / SpaCy**
* **Pandas & NumPy**

### How It Works

1. The user uploads a resume.
2. The system extracts text from the document.
3. NLP techniques analyze skills, keywords, and experience.
4. The resume is compared with a job description.
5. A matching score and improvement suggestions are generated.

### Applications

* Resume optimization for job seekers
* Recruitment pre-screening
* Career guidance platforms
* HR automation tools
